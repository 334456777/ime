# SRT 智能修正工具

## 项目意义

这个工具是我在视频剪辑工作中开发的专用工具，主要解决SRT字幕文件中的拼音输入错误问题。在视频剪辑过程中，字幕往往是通过语音识别或手动输入生成的，经常会出现拼音错误，比如"太傻"被误写成"泰尔"、"密度"被写成"私密度"等。这些错误如果不修正，会影响字幕的专业性和可读性。

工具的核心价值在于：
- **自动化纠错**：基于预定义的关键词列表，自动检测和修正常见的拼音错误
- **智能过滤**：通过黑名单机制，避免对特定专业术语进行不必要的修改
- **灵活配置**：支持手动添加替换规则，适应不同的内容需求
- **批量处理**：可以快速处理整个SRT文件，提高工作效率

这个工具特别适用于处理涉及特定领域词汇的视频内容，比如哲学、灵性、科学等主题的字幕修正。

## 功能特性

- **拼音纠错**：基于关键词列表自动检测和修正拼音错误
- **黑名单过滤**：避免对特定词语进行不必要的修正
- **手动替换规则**：支持自定义的错误词到正确词的映射
- **SRT 文件处理**：专门针对 SRT 字幕文件格式进行处理，保持时间戳和格式不变
- **命令行操作**：简单易用的命令行界面，支持批量处理

## 使用方法

### 基本用法

工具支持两种主要功能模式：

#### 功能1: 智能修正字幕 (-i 参数)

这是工具的核心功能，用于自动修正SRT字幕文件中的拼音错误。

**用法：**
```bash
ime -i <输入文件.srt> <输出文件.srt>
```

**示例：**
```bash
ime -i input.srt corrected_output.srt
```

**工作流程：**
1. 自动读取同目录下的 `config.txt` 配置文件
2. 解析输入的SRT文件
3. 对每个字幕块的内容进行智能修正：
   - 首先应用手动替换规则（ManualFixes）
   - 然后进行拼音相似度匹配和自动纠错
4. 输出修正后的字幕文件
5. 显示修正统计信息（修改了多少字幕块、具体的替换情况）

**输出示例：**
```
>> 已加载配置: config.txt
>> 读取文件: input.srt
>> 正在执行修正...共修改了 5 个字幕块
>> 写入输出文件: corrected_output.srt

修正统计:
泰尔 -> 太傻: 2 次
私密度 -> 四密度: 3 次
```

#### 功能2: 转移字幕内容 (-t 参数)

用于将部分字幕文件的内容应用到完整字幕文件中，常用于字幕校对工作流程。

**用法：**
```bash
ime -t <部分字幕.srt> <完整字幕.srt> <输出文件.srt>
```

**示例：**
```bash
ime -t partial_corrections.srt full_subtitle.srt final_output.srt
```

**部分字幕文件格式：**
部分字幕文件使用特殊格式，以 `[序号]` 开头：
```
[1] 00:00:00,260 --> 00:00:04,510
修正后的第一段字幕内容

[5] 00:00:20,150 --> 00:00:24,890
修正后的第五段字幕内容
```

**工作流程：**
1. 读取部分字幕文件（带序号格式）
2. 读取完整字幕文件
3. 根据序号匹配，将部分字幕的内容替换到完整字幕的对应位置
4. 生成最终的输出文件
5. 显示转移统计信息

**输出示例：**
```
>> 读取部分字幕文件: partial.srt
>> 读取完整字幕文件: full.srt
>> 正在转移字幕内容...共转移了 3 个字幕块
>> 写入输出文件: output.srt
```

### 详细工作流程

#### 智能修正模式的工作原理

1. **配置加载**：读取 `config.txt` 中的关键词、黑名单和手动规则
2. **文件解析**：将SRT文件解析为字幕块结构
3. **内容分析**：对每段字幕文本进行以下处理：
   - **手动替换**：优先执行 ManualFixes 中的精确匹配替换
   - **拼音分析**：将文本转换为拼音序列
   - **相似度计算**：使用编辑距离算法计算与关键词的拼音相似度
   - **阈值判断**：根据词长度应用不同的相似度阈值（2字词95%，3字词80%，其他70%）
   - **黑名单过滤**：跳过黑名单中的词
   - **声调辅助**：考虑声调匹配度进行额外奖励
4. **结果输出**：生成修正后的SRT文件，保持原有时间戳格式

#### 转移模式的工作原理

1. **格式识别**：识别部分字幕中的 `[序号]` 格式
2. **序号匹配**：将部分字幕按序号映射到完整字幕的对应位置
3. **内容替换**：精确替换匹配序号的字幕内容
4. **格式保持**：完整保留原有的时间戳和SRT结构

### 配置文件的详细说明

`config.txt` 是工具的核心配置文件，格式如下：

#### [UserKeywords] 部分
这是关键词列表，每行可以包含一个或多个词。这些词用于拼音匹配和纠错。
- 工具会将输入文本与这些关键词进行拼音相似度比较
- 如果发现相似度高的拼音错误，会自动修正
- 支持多词组合，如"无限维创之匙"

#### [BlacklistWords] 部分
黑名单词列表，这些词不会被修正。
- 用于保护专业术语或特定表达
- 例如密度相关的专业术语："第一密度 第二密度"等

#### [ManualFixes] 部分
手动替换规则，格式：`错误词=正确词`
- 这些规则优先级最高，会在自动纠错之前执行
- 例如：`泰尔=太傻`，`私密度=四密度`

### 配置示例

```
# SRT 智能修正工具配置文件

[UserKeywords]
太傻天书 无限维创 意识密度 九维宇宙
四密度 复合叠加 波叠加 问答环节
太傻 密度 维度 意识 魔法

[BlacklistWords]
第一密度 第二密度 第三密度 第四密度

[ManualFixes]
泰尔=太傻
私密度=四密度
多四密度=第四密度
```

### 注意事项

- **备份重要性**：工具会自动备份原文件，请放心使用
- **格式保持**：SRT 文件的时间戳和结构会被完整保留
- **人工复核**：虽然算法智能，但仍建议人工检查修正结果
- **配置更新**：根据使用情况不断完善 config.txt 中的规则
- **编码支持**：支持UTF-8编码的SRT文件，自动处理BOM
